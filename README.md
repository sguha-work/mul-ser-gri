## initialize ⇒ <code>Number</code>
- This function is the constructor of LifeGrid

**Kind**: global variable  
**Returns**: <code>Number</code> - - The number of effected rows  

| Param | Type | Description |
| --- | --- | --- |
| gridId | <code>String</code> | The unique id of grid |
| containerId | <code>String</code> | The container DOM id where the grid is going to be populated |
| attributes | <code>Object</code> | The attributes object, defines all settings |
| data | <code>Object/Array</code> | The data which is going to be displayed in grid |
